# **App Name**: AdminWeb

## Core Features:

- Admin Authentication: Secure login functionality to restrict access to authorized personnel only.
- Role & Permissions: Define roles (e.g., Admin, Supervisor) and assign granular access rights.
- Branch Management: CRUD operations for branch information (name, address, phone, hours) and immediate sync to User App.
- POS365 Integration: Secure management of POS365 API credentials (key masking and encryption). Logging of config changes.
- Automated Notifications: Configure triggers for automated messages on events (e.g., tier upgrade, points expiring), push them using API.
- Activity Audit Log: Centralized audit log of important Web Admin actions, with filtering and detailed change tracking.
- Customer Segmentation Tool: allowing user to create segments by combining one or more conditions of customer data such as behavioral, demographic, membership, stamp point, last order date, order frequency, total spend
- Menu and Category Manager: Enables the generation and management of menu items and categories in the admin UI. Realtime update to users.
- Advertising Banner Manager: Enable management of advertisement banners to advertise things to end-users of the mobile app.
- Notification Campaign Management: Configure notification campaigns based on different factors.
- Loyalty and Member Tier Configuration: Allows for MTPS config based on members.
- Loyalty Stamp Configuration: Configuration for collecting loyalty stamps
- Loyalty Rewards configuration: Manages loyalty rewards with various methods.
- Order Manager: Manage various orders made with the system.
- Operational and Customer Response Reports: Generates report data about performance and reports on operations.
- Booking management and lookups: Bookings management, viewing different statuses.

## Style Guidelines:

- Primary color: A calm blue (#73A6C2) to invoke feelings of trust and professionalism.
- Background color: Light, desaturated blue (#E0E8EF), to maintain a clean and spacious feel.
- Accent color: A vibrant orange (#D9894D), to make important elements like call-to-action buttons stand out without disrupting the balance.
- Font pairing: 'Inter' (sans-serif) for headings, and 'PT Sans' (sans-serif) for body text. These sans-serif fonts complement each other in their readability and clean lines, ideal for UI/UX design. This combination creates a harmonious visual hierarchy.
- Code font: 'Source Code Pro' for any displayed code snippets.
- Simple, geometric icons for key functions and data visualization.
- Clean, grid-based layout to ensure information is presented clearly and accessibly.
- Subtle transitions and animations to provide feedback on user actions.
- Phong cách chung: Hướng đến sự hiện đại, sạch sẽ và chuyên nghiệp. Giao diện cần thể hiện sự tin cậy và tập trung vào dữ liệu, phù hợp với môi trường doanh nghiệp.
- Trải nghiệm người dùng: Thân thiện và tập trung vào hiệu quả. Giảm thiểu số lần nhấp chuột cho các tác vụ lặp đi lặp lại (như kiểm duyệt). Luồng công việc phải logic và dễ đoán cho người dùng quản lý.
- Bảng màu: Sử dụng các gam màu trung tính (xám nhạt, trắng) làm nền chủ đạo để tạo cảm giác thoáng đãng. Kết hợp với màu thương hiệu (ví dụ: xanh dương) cho các yếuTố điều hướng chính. Sử dụng màu sắc chức năng một cách nhất quán (ví dụ: Xanh lá cho 'Hoàn thành', Vàng cho 'Chờ duyệt', Đỏ cho 'Quá hạn').
- Typography: Sử dụng một font chữ sans-serif hiện đại, dễ đọc (ví dụ: Inter, Lato, Roboto). Phân cấp thông tin rõ ràng bằng cách sử dụng kích thước, độ đậm và màu sắc của chữ.
- Iconography: Sử dụng một bộ icon đồng nhất, tối giản và dễ hiểu để biểu thị các chức năng, giúp người dùng định vị nhanh hơn.
